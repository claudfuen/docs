<Update label="October 2024">
## Workflows 2.0
Public release of Workflows 2.0, a complete rewrite of the Leap platform which now supports building more sophisticated workflows.

### Key Features

- **Integrations**: Added support for Slack and GitHub
- **Workflow Scheduling**: Added support for scheduled workflows using cron expressions
- **Conditionals**: Added support for conditionals to add branching logic to your workflows
- **Loops**: Added support for loops to repeat actions in your workflows

### Initial integrations

- **Slack**: New triggers for Slack messages and reactions
- **GitHub**: New triggers for GitHub pull requests and comments
- **Exa**: Added integration with Exa for semantic search and embeddings
- **Fal.ai**: Added integration with Fal.ai for image generation and processing
- **OpenAI**: Added integration with OpenAI for text generation, embeddings, and image generation
- **Anthropic**: Added integration with Anthropic's Claude for text generation and analysis

</Update>
